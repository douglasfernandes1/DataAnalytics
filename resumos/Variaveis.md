# O que são variáveis?

## Tipos, o que são?

Os tipos são as definições das características e comportamentos de um objeto para um interpretador. 

Tipos principais:


Tipo | caracteristica
----- | -------
Texto | Str
Numérico | int, float, complex
Sequência| list, tuple, range
Mapa| dict
Coleção| set, frozenset
Booleano| bool
Binário| bytes, bytearray
